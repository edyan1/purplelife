Purple Life is a game based
on Conway’s Game of Life.
It borrows some elements from
Conway’s game but also expands
upon it by adding new features
such as having a set objective
and being able to advance from
one level to another. A user
is also able to create their own
custom levels using our own
proprietary level maker.
